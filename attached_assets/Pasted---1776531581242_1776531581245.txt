Создай полноценный веб-сайт приложения для именных свадебных приглашений с современной, романтичной, премиальной подачей. Сайт должен выглядеть дорого, стильно, нежно и эмоционально. Основная тема — свадьба, с участием жениха и невесты как ключевых персонажей визуального стиля и мини-игры.

Общая идея

Это сайт свадебного приглашения, где каждое приглашение персонализировано под конкретного гостя. Для каждого гостя должна генерироваться индивидуальная ссылка. После перехода по ссылке пользователь не сразу видит приглашение — сначала он проходит небольшую тематическую 3D-игру, и только после успешного прохождения открывается красивое именное приглашение.

Цель проекта

Нужен MVP, но визуально качественный, аккуратный и готовый к использованию. Код должен быть чистым, логичным, с понятной структурой, чтобы потом можно было доработать.

Технический стек

Используй стек, который хорошо подходит для Replit и быстро запускается:
	•	Frontend: React + Vite
	•	Backend: Node.js + Express
	•	База данных: SQLite или Supabase (если проще для Replit, можно SQLite для MVP)
	•	UI: современный адаптивный дизайн
	•	Для карты: встроить Google Maps или OpenStreetMap
	•	Для 3D-игры: Three.js или React Three Fiber
	•	Для стилей: Tailwind CSS
	•	Для анимаций: Framer Motion
	•	Для генерации уникальных ссылок: UUID или slug
	•	Для деплоя и запуска всё должно быть совместимо с Replit

4. Индивидуальные приглашения

Каждое приглашение открывается по уникальной ссылке, например:
/invite/anna-ivanova-ab12cd

На странице приглашения:
	•	Если игра ещё не пройдена, сначала показывается экран приветствия и кнопка “Начать”
	•	После этого запускается 3D-игра
	•	Только после прохождения игры открывается приглашение
	•	В приглашении должно быть персональное обращение:
	•	“Дорогая Анна Иванова”
	•	“Дорогой Иван Петров”
	•	“Дорогие Анна и Сергей”
	•	Обращение должно подставляться автоматически из данных гостя

5. 3D-игра перед открытием приглашения

Нужна простая, но эффектная тематическая 3D-игра, связанная со свадьбой.

Идея игры:

Сделай романтическую 3D-мини-игру, где игрок должен помочь жениху и невесте встретиться или собрать свадебные символы, чтобы открыть приглашение.

Вариант механики:

Игрок управляет сердцем / кольцом / персонажем и должен собрать 5–7 объектов:
	•	сердца
	•	цветы
	•	кольца
	•	звёзды
	•	лепестки

Либо:
жених идёт к невесте через красивую романтическую сцену, и пользователь помогает пройти путь, обходя препятствия.

Требования к игре:
	•	Игра должна быть 3D
	•	Тематика свадьбы
	•	В игре должны присутствовать жених и невеста
	•	Должна быть не слишком сложной
	•	Должна работать в браузере
	•	После успешного прохождения показывать красивую анимацию открытия приглашения
	•	Если пользователь проиграл — можно попробовать ещё раз
	•	На мобильных тоже должна быть приемлемая работа
	•	Не делать тяжёлую AAA-графику, нужен лёгкий MVP с приятной стилизацией

Визуальная стилистика игры:
	•	романтичная
	•	светлая
	•	пастельные тона
	•	мягкое освещение
	•	свадебные элементы
	•	воздушные частицы, лепестки, блёстки
	•	музыка по желанию, но можно предусмотреть кнопку включения/выключения
