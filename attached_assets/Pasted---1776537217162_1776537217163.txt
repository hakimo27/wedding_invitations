Доработай уже существующий проект свадебного сайта-приглашения. Не создавай новый проект с нуля. Работай поверх текущего кода.

Новая задача

Подготовь проект к нормальному production deployment на обычный сервер/VPS. Нужно, чтобы проект можно было собрать, распаковать на сервере, прописать env, домен, включить SSL и быстро запустить.

Мне нужна не только разработческая версия, а именно готовность к развёртыванию.

3. Добавить env-конфигурацию

Сделай полноценную поддержку .env.

Создай:
	•	.env.example

В env должны быть понятные переменные:
	•	NODE_ENV=production
	•	PORT=3000
	•	APP_BASE_URL=https://your-domain.com
	•	ADMIN_PASSWORD=change_me
	•	DB_PATH=./data/app.db
	•	SSL_EMAIL=your@email.com если нужно для certbot-инструкции
	•	CORS_ORIGIN=https://your-domain.com
	•	другие необходимые переменные, если проект их использует

Важно:
	•	все абсолютные ссылки должны собираться из APP_BASE_URL;
	•	копирование invite-ссылки должно брать URL из env/base url;
	•	Telegram share должен использовать полный production URL;
	•	Open Graph / social preview тоже должны использовать production domain.
